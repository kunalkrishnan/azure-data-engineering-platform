# Azure End-to-End Data Engineering Platform

## Overview

This project demonstrates a scalable Azure data engineering pipeline using modern cloud technologies.

The pipeline ingests raw data, transforms it using PySpark, and loads it into a data warehouse for analytics.

## Architecture

Data Source → Azure Data Factory → Azure Data Lake → Databricks → Delta Lake → Azure Synapse → Power BI

## Technologies Used

Azure Data Factory  
Azure Data Lake Gen2  
Azure Databricks  
PySpark  
Delta Lake  
Azure Synapse Analytics  
SQL  

## Project Structure

architecture – Architecture documentation  
data – Sample data for pipeline  
databricks – PySpark transformation scripts  
sql – Data warehouse schema
